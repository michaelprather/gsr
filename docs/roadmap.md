# Roadmap

This document contains ideas for future development. Skip it when writing stories.

## Gameplay & Rules
	•	Allow rounds to be played out of order
	•	Built-in scoring calculator
	•	Rules and scoring reference within the app
	•	Tiebreaker support

## Player & Game Management
	•	Add or remove players mid-game
	•	Save player profiles
	•	Multiple saved games
	•	Game history and statistics

## Sharing & Sync
	•	Shareable, tokenized game links
	•	View-only and edit permissions
	•	Real-time multi-device sync
	•	Optional cloud storage

## UI & Experience
	•	Dark mode
	•	Animations and enhanced feedback
	•	Custom themes
	•	Accessibility enhancements

## Data & Analytics
	•	Aggregate stats across games
	•	Average scores per round
	•	Win history per player
